## Run Locally

Clone the project

```bash
  git clone https://github.com/Lokesh01/React_TS_Select_options
```

Go to the project directory

```bash
  cd React_TS_Select_options
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
